# buzzer
	- 원하는 주파수 값을 인가하여 소리를 내는 소자
	- 극성이 존재하므로 연결시 주의 필요

## 참고자료
![scale_hz](./scale_hz.png)

## 필요 하드웨어
	- buzzer
	- Arduino UNO
	- UNO cable
	- breadboard
	- M-M cable(2ea)

## 연결
	- D7단자 대신 다른 디지털 핀에 연결해도 무방
|buzzer|Audino UNO|
|--|--|
|+단자|D7(변경 가능)|
|-단자|GND|
